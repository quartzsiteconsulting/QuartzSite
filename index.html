Portfolio

Brampton.ca SharePoint Upgrade

Supported the comprehensive migration of Brampton’s municipal website, encompassing over 2500 webpages, from SharePoint 2010 to the latest SharePoint Server Edition.

This project involved detailed planning, stakeholder engagement, and the use of advanced migration tools to ensure a seamless transition while enhancing accessibility, functionality, and user experience. The successful upgrade significantly improved the site’s performance and maintainability, aligning it with contemporary web standards and city governance needs.

Flynn Group of Companies SharePoint Upgrade

Oversaw the migration of 30+ branches and 15+ departments from SharePoint On-Premise to SharePoint Online. This comprehensive upgrade encompassed not just migration but also extended to providing ongoing support and creating extensive resources including technical documentation, training modules, video tutorials, and change management strategies. This project streamlined operations, enhanced collaboration, and ensured a smooth transition with minimal disruption, equipping all levels of the organization with the tools and knowledge needed for effective utilization of the new system.
